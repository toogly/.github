## Hey, we're Toogly

**Time tracking for developers who live in the terminal.**

Toogly is a terminal-first time tracker built with Go and [Bubble Tea](https://github.com/charmbracelet/bubbletea). It's fast, offline-first, and syncs when you're ready.

### What we're building

- **TUI client** &mdash; a beautiful terminal interface for tracking time without leaving your workflow
- **Backend API** &mdash; Go-powered sync server with PostgreSQL
- **Web dashboard** &mdash; reports, billing, and team management (coming soon)

### Tech stack

`Go` `Bubble Tea` `PostgreSQL` `SQLite` `net/http`

No frameworks. Standard library where possible. One language across the entire stack.
