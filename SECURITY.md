# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.** Instead, email security concerns to the maintainers or use [GitHub's private vulnerability reporting](https://github.com/toogly/.github/security/advisories/new).

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

## Supported Versions

We provide security updates for the latest release only.
